# Neuro-Fuzzy Student Performance Prediction

## 🔍 Overview

This project implements a **Neuro-Fuzzy (ANFIS-inspired) system** to predict student performance based on academic indicators such as attendance, assignment scores, and test marks.

It combines the interpretability of **fuzzy logic** with the learning capability of **neural networks**, creating a hybrid intelligent system.

---

## ⚙️ Key Features

* Fuzzification using bell-shaped membership functions
* Rule-based fuzzy inference system
* ANFIS-inspired neural network architecture
* Hybrid prediction (fuzzy + neural ensemble)
* Performance evaluation using confusion matrix and classification metrics

---

## 🧠 Methodology

1. Generate dataset (attendance, assignment, test scores)
2. Apply fuzzification (Poor, Average, Good)
3. Define fuzzy rules (Takagi-Sugeno type)
4. Build ANFIS-inspired neural network
5. Train and evaluate the model
6. Combine fuzzy and neural predictions
